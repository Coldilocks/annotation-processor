# annotation-processor


## Build step
1. Build annotation
- `cd factory/annotation`
- run `mvn clean install`

2. Build factory
- `cd factory`
- run `mvn clean install`

3. Build sample
- `cd factory-sample/pizzastore`
- run `mvn -q  package exec:java`
